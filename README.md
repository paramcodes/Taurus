# Taurus
Taurus is a lightweight AI assistant web app powered by Next.js and Google GenAI. It provides a clean chat experience with streaming-style UX patterns, rich Markdown support (including tables, math, and code), syntax highlighting with one-click code copy, and local chat history persistence so conversations stay available across sessions.

## ⚡ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/paramcodes/Taurus.git
cd Taurus
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key
```

---

### 4. Run the development server

```bash
npm run dev
```

App will run at:

```
http://localhost:3000
```
